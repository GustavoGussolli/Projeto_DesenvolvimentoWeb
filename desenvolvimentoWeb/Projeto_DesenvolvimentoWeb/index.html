Títulos servem para definir o tema de um site. Existem seis níveis de títulos

Podemos destacar trechos de texto para enfatizar uma ideia!

  • Uma lista não ordenada não se importa com ordem
  • UL siginifica "unordered list"
  • LI siginifica "List Item"
  1. Uma lista ordenada é uma lista que possue algum tipo de ordem númerica
  2. Seu funcionamento é exatamente como uma ul
  3. OL significa "Ordered List"
